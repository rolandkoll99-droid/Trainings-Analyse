📱 Trainingsanalyse-App – Kurzanleitung
🎯 Was macht die App?

Die App hilft Ihnen, Ihr Stockschießen-Training systematisch zu erfassen und auszuwerten. Sie können pro Training bis zu 12 Spiele mit je 6 Kehren dokumentieren.
🚀 So funktioniert's
1. Spieler anlegen

    Geben Sie Ihren Namen ein

    Klicken Sie auf "💾 Profil speichern"

    Ihre Daten werden automatisch im Browser gespeichert

2. Training eingeben

    Wählen Sie ein Spiel (1-12) aus

    Pro Kehre (1-6) wählen Sie einen Schuss:

        🎯 Maß (grün)
        🏒 Stock (blau)
        ❌ Fehler (rot)

    Wählen Sie die verwendete Platte pro Kehre aus

    Klicken Sie "✅ Fertig" wenn alle 6 Kehren gespielt sind

3. Analyse ansehen

    Wechseln Sie zum Tab "📊 Analyse"

    Sie sehen sofort:

        Trefferquote in %
        Verteilung von Maß-, Stock- und Fehlschüssen
        Beste/schlechteste Platte
        Form-Trend der letzten Spiele
        Beste Serie

4. Daten speichern & exportieren

    💾 In Datenbank speichern → in die Cloud (Google Sheets)
    💾 Lokal speichern → im Browser
    📄 CSV Export → als Tabelle für Excel
    📄 PDF Export → als ausdruckbares Dokument

📊 Was bedeuten die Farben?
Farbe	Bedeutung
🟢 Grün	Trefferquote ≥ 80% – sehr gut
🟡 Gelb	Trefferquote 60-79% – okay
🔴 Rot	Trefferquote < 60% – Verbesserung nötig
💡 Tipps für die Nutzung
Tipp	Beschreibung
📱 Tablet	Die App ist für iPads optimiert – große Buttons zum Tippen
💾 Automatisch	Ihre Daten werden beim Wechseln automatisch gespeichert
🔄 Spieler wechseln	Einfach Namen ändern und "Laden" klicken
📈 Fortschritt	Im Tab "Historie" sehen Sie Ihre Entwicklung
📤 Teilen	CSV oder PDF exportieren und an Trainer senden
🔧 Technische Hinweise
Hinweis	Beschreibung
🌐 Internet	Für Cloud-Speicherung (Google Sheets) wird Internet benötigt
💾 Lokal	Daten bleiben auch ohne Internet im Browser gespeichert
🗑️ Löschen	Klicken Sie auf "🧹 Leeren" oder "🗑️" um Daten zu entfernen
📂 Import	Sie können gespeicherte CSV-Dateien wieder laden
📞 Fragen?

Bei Problemen oder Fragen wenden Sie sich an Ihren Trainer oder den App-Administrator.
